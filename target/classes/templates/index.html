Sequence of Actions Performed by the Automation Script:

Test Case: 01

  • Step-1:

    Open the JPetshop in the Browser.

  • Step-2:

    Verify Browser opened the correct URL.

  • Step-3:

    Collect list of options that present both in the vertical and Horizontal of the page.

  • Step-4:

    Verify both the list are not empty.

  • Step-5:

    And also Verify both the list are same.

  • Step-6:

    Store that list of options in the Excel sheet.

Test Case: 02 (Depends on test case - 1)

  • Step-1:

    Open the JPetshop in the Browser.

  • Step-2:

    Verify Browser opened the correct URL.

  • Step-3:

    Collect list of options from excel sheet that created with the help of previous test case.

  • Step-:4

    With the help of the collected list enter into the particular option.

  • Step-5:

    Collect all Product ID and Product Name.

  • Step-6:

    Store the collected data into the excel sheet.

  • Step-7:

    Repeat the process for every options in the list.
